♿ Declaración de Accesibilidad 🌟

📅 Fecha de Vigencia: 1 de enero de 2025

🌟 1. Nuestro Compromiso con la Accesibilidad

En Torres Granite, estamos comprometidos a garantizar que nuestro sitio web sea accesible para todos, independientemente de sus capacidades. Nos esforzamos por ofrecer una experiencia inclusiva para todos los usuarios. 🌐

✅ 2. Estándares de Accesibilidad

Nuestro sitio web está diseñado para cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1, Nivel AA. Estas pautas aseguran que nuestro contenido sea accesible para personas con diversas discapacidades, incluidas discapacidades visuales, auditivas y cognitivas. 🛠️

🔧 3. Características de Nuestro Sitio Accesible

  • 📖 Alternativas de texto para contenido no textual.
  • ⌨️ Navegación con teclado para todos los elementos interactivos.
  • 🖥️ Diseño responsivo que se adapta a diferentes tamaños de pantalla y tecnologías de asistencia.
  • ⚙️ Contrastes de color que aseguran que el texto y las imágenes sean legibles.

🔄 4. Mejoras Continuas

Continuamente estamos mejorando la accesibilidad de nuestro sitio web. Los comentarios de los usuarios son una parte vital de este proceso y agradecemos las sugerencias sobre cómo podemos mejorar. 💡

Si encuentra algún problema de accesibilidad mientras utiliza nuestro sitio web, contáctenos. Estamos aquí para ayudarle y abordar cualquier inquietud que pueda tener. 💬
